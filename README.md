## routing


### View: `view`

- ゲームの情報を知らない
  - `data` から与えられた、盤面情報を描画するだけ

### Model: `data`

- ゲーム全体のデータを保持
  - 初期盤面配置
  - 盤面情報
    - 毎回0手目から、指定手目までループして構築
  - 初手から終局までの指し手情報
    - 初期盤面から、指し手情報を盤面情報を更新
  - 持ち駒
    - いい感じで、取得や使った情報を更新

### Control: `push`
    
- 何手目かの指示を出す
  - スライダーやボタン
  - ゲーム開始時は0手目
- 何手目を指示しているかの情報を保持



```
view:      data のn手目   |              data のn+1手目
           情報に書き換え |              情報に書き換え
          /               |             /
data:   0..n と0 からloop |             0...n+1 とloop
        n手目まで、進めて |             0からn+1 まで進めて
        盤面情報を保持    |             盤面情報を保持
       /                  |            /
push: n手目               |push から   n+1手目
     (指示番号を保持=n)   |更新指示    (指示番号を保持=n+1)

```
